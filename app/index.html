AHB - Farm Budgets App (Demo)

This is only a prototype to demo the applications proposed functionality. This application is not yet intended to perform any real world analysis about crop budgets at this point.
The Advanced Hardwood Biofuels Northwest project, AHB-NW investigates using poplar as a feedstock source for drop in gasoline replacement in the Pacific Northwest. UC Davis, is part of the sustainability group and created this appication for crop budget modeling.

Instructions

Select Location

First use the map to find a location. You can simply click the map to select a location as well. Once a location is set, the app will lookup all available crops for the location.

Select Crop and Size

On the crop page, you will be shown a list of crops for the selected location. Select the crop you wish to budget as well as the farm size.

View Budget

Once you have selected a location, crop and farm size, you will be shown the default budget for the crop and location. Click the arrows next to any budget category to see detailed information about the budget. You can then modify any budget parameter you wish. This includes adding custom items, using the 'Add Custom' button at the top right. You can also remove items using the '- Remove' button next to any budget item. The application will recalulate as soon as modifications are made.
