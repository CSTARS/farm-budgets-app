AHB - Farm Budgets App (v0.2)

This is a early demo of the applications proposed functionality. This application is not yet intended to perform any real world analysis about crop budgets at this point.
The Advanced Hardwood Biofuels Northwest project, AHB-NW investigates using poplar as a feedstock source for drop in gasoline replacement in the Pacific Northwest. UC Davis, is part of the sustainability group and created this appication for crop budget modeling.

Instructions

Select Location

First use the map to find a location. You can simply click the map to select a location as well. Once a location is set, the app will lookup all available crops for the location.

Select Crop and Size

On the crop page, you will be shown a list of crops for the selected location. Select the crop you wish to budget as well as the farm size.

View Budget

Once you have selected a location, crop and farm size, click 'View Budget' to see the default budget for the crop and location. Click the arrows next to any budget phase to see operation and material details about the budget. You can then modify any budget parameter you wish. The application will recalulate as soon as modifications are made.
